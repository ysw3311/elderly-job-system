# 노인 일자리 추천 시스템 - 사용 가이드

## 📦 프로젝트 설치 및 실행

### 1단계: 프로젝트 압축 해제
```bash
# 다운로드 받은 파일 압축 해제
tar -xzf elderly-job-system.tar.gz

# 또는 zip 파일인 경우
unzip elderly-job-system.zip
```

### 2단계: 프로젝트 디렉토리 이동
```bash
cd elderly-job-system
```

### 3단계: 의존성 설치
```bash
npm install
```

설치되는 주요 패키지:
- react & react-dom (v18.3.1)
- lucide-react (아이콘 라이브러리)
- vite (빌드 도구)
- tailwindcss (스타일링)

### 4단계: 개발 서버 실행
```bash
npm run dev
```

서버가 시작되면 자동으로 브라우저가 열립니다.
주소: http://localhost:3000

### 5단계: 프로덕션 빌드 (선택사항)
```bash
npm run build
```

빌드된 파일은 `build/` 디렉토리에 생성됩니다.

## 🎯 시스템 테스트 시나리오

### 시나리오 1: 기업의 공고 등록부터 정부 승인까지

#### Step 1: 기업으로 로그인
1. 메인 페이지에서 "로그인" 클릭
2. 빠른 로그인: "현대백화점으로 로그인" 버튼 클릭
   - 또는 직접 입력: ID=`company_hyundai`, PW=`comp123`

#### Step 2: 새 공고 등록
1. 대시보드에서 "새 공고 등록" 버튼 클릭
2. 다음 정보 입력:
   - 공고 제목: "매장 정리 보조"
   - 업무 내용: "상품 진열 및 매장 정리"
   - 근무지: "서울 중구"
   - 고용형태: 시간제
   - 시급: 12500
   - 근무일: "월, 수, 금"
   - 근무시간: "10:00-15:00"
   - 근무 기간: "3개월"
   - 지원 마감일: (오늘부터 1달 후)
3. "공고 등록" 버튼 클릭

#### Step 3: 상태 확인
- 등록한 공고 탭에서 상태 확인
- 노란색 배지: "승인 대기" 표시

#### Step 4: 로그아웃
- 우측 상단 "로그아웃" 버튼 클릭

---

### 시나리오 2: 정부의 공고 승인

#### Step 1: 정부 관리자로 로그인
1. "로그인" 클릭
2. 빠른 로그인: "정부로 로그인" 버튼 클릭
   - 또는 직접 입력: ID=`gov_admin`, PW=`gov123`

#### Step 2: 대기 중인 공고 확인
1. "승인 대기 중인 공고" 섹션 확인
2. 방금 등록한 "매장 정리 보조" 공고 확인

#### Step 3: 공고 검토
1. "검토하기" 버튼 클릭
2. 모달 창에서 공고 내용 확인:
   - 공고 제목
   - 기업명
   - 업무 내용
   - 근무 조건 (급여, 근무일, 시간 등)

#### Step 4: 승인
1. "승인" 버튼 클릭
2. 승인된 공고 섹션으로 이동 확인

#### Step 5: 로그아웃

---

### 시나리오 3: 노인의 일자리 검색 및 지원

#### Step 1: 노인 회원으로 로그인
1. "로그인" 클릭
2. 빠른 로그인: "박철수로 로그인" 버튼 클릭
   - 또는 직접 입력: ID=`senior_park`, PW=`senior123`

#### Step 2: 추천 일자리 확인
1. 대시보드에서 "추천 일자리" 탭 (기본 선택)
2. 정부 승인된 공고들 확인
3. 각 공고에는 녹색 "정부 승인" 배지 표시

#### Step 3: 공고 상세 확인
- 공고 제목, 기업명
- 업무 내용
- 근무지, 급여, 근무일, 시간
- 근무 기간

#### Step 4: 지원하기
1. 원하는 공고의 "지원하기" 버튼 클릭
2. 확인 모달에서 정보 재확인
3. "지원하기" 버튼 클릭하여 최종 지원

#### Step 5: 지원 내역 확인
1. "지원 내역" 탭 클릭
2. 지원한 공고 확인
3. 상태: "검토중" (파란색 배지)

---

### 시나리오 4: 기업의 지원자 승인

#### Step 1: 기업으로 다시 로그인
1. 로그아웃 후 "로그인"
2. 현대백화점으로 로그인

#### Step 2: 지원자 관리 탭 이동
1. "지원자 관리" 탭 클릭
2. 박철수의 지원서 확인

#### Step 3: 지원자 정보 확인
- 지원자 이름
- 지원한 공고
- 지원일

#### Step 4: 채용 결정
옵션 3가지:
1. **승인 (채용)**: 즉시 채용 확정
2. **면접 예정**: 면접 일정 조율
3. **거절**: 불합격

"승인 (채용)" 버튼 클릭

#### Step 5: 자동 이력 등록 확인
- 승인 즉시 자동으로 근무 이력 생성
- 상태가 "승인됨 (근무 시작)"으로 변경

---

### 시나리오 5: 노인의 이력 관리 확인

#### Step 1: 노인으로 다시 로그인
1. 로그아웃 후 박철수로 로그인

#### Step 2: 채용 결과 확인
1. "지원 내역" 탭에서 상태 확인
2. 녹색 배지: "승인됨 (근무 시작)"
3. 안내 메시지: "채용되었습니다! 이력관리에서 근무 내역을 확인하실 수 있습니다."

#### Step 3: 대시보드 알림 확인
- 페이지 상단에 녹색 알림 박스 표시:
  ```
  ✓ 현재 근무중입니다!
  현대백화점 - 매장 정리 보조
  근무 시작일: 2025-XX-XX
  ```

#### Step 4: 이력 관리 확인
1. "이력 관리" 탭 클릭
2. 근무 이력 자동 등록 확인:
   - 회사: 현대백화점
   - 직책: 매장 정리 보조
   - 근무 시작일: (자동 기록)
   - 상태: "근무중" (녹색 배지)
   - "검증된 이력" 표시 (파란색 체크마크)

## 🔄 데이터 초기화 방법

### 방법 1: 브라우저 개발자 도구 사용
1. F12 키를 눌러 개발자 도구 열기
2. "Application" 또는 "애플리케이션" 탭 클릭
3. 좌측 메뉴에서 "Local Storage" 클릭
4. `localhost:3000` 선택
5. 모든 항목 선택 후 Delete 키

### 방법 2: 콘솔에서 명령어 실행
1. F12 개발자 도구 열기
2. "Console" 탭 클릭
3. 다음 명령어 입력:
```javascript
localStorage.clear()
```
4. 페이지 새로고침 (F5)

## 🐛 문제 해결

### 문제 1: npm install 실패
**해결책:**
```bash
# 캐시 정리
npm cache clean --force

# node_modules 삭제 후 재설치
rm -rf node_modules
npm install
```

### 문제 2: 포트 3000이 이미 사용 중
**해결책:**
```bash
# vite.config.js에서 포트 변경
# server.port를 3001로 변경
```

### 문제 3: 데이터가 저장되지 않음
**원인:** 브라우저 시크릿 모드 사용
**해결책:** 일반 브라우저 창에서 실행

### 문제 4: 스타일이 적용되지 않음
**해결책:**
```bash
# Tailwind CSS 재빌드
npm run dev
```

## 📝 개발자 노트

### 주요 파일 설명

- **App.jsx**: 전체 앱의 라우팅 및 상태 관리
- **storage.js**: LocalStorage CRUD 작업 처리
- **MainPage.jsx**: 공개 랜딩 페이지
- **Login.jsx**: 인증 페이지
- **SignUp.jsx**: 회원가입 페이지
- **GovernmentDashboard.jsx**: 정부 관리자 인터페이스
- **CompanyDashboard.jsx**: 기업 관리 인터페이스
- **SeniorDashboard.jsx**: 노인 회원 인터페이스

### 데이터 흐름

1. **초기화**: storage.js의 `initializeStorage()` 호출
2. **로딩**: App.jsx의 useEffect에서 데이터 로드
3. **업데이트**: useState의 setter 함수로 상태 변경
4. **저장**: useEffect로 변경 감지 및 LocalStorage 저장
